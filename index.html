Fusion X
About Fusion X

Fusion X is modern multi-feature desktop application developed with the aim to integrate multiple applications into one, whether it's your favorite media, or you want to go maps, Fusion X is there at your service. Fusion X is python based, with the rich set of python modules induced in the application development process, Fusion X utilizes multiple python modules. Being a Tkinter GUI based application, it is easy to use, develop, and is cross platform with portable compact code design. All the above functionalities works in sync to provide an extensible area to explore around with.
